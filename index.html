Enunciate.io

Submission for VTHacks, February 2019

Enunciate.io allows users to read our virtual children's books out loud to their computer and get real-time feedback on the accuracy of their word pronunciation. We used a Speech Recognition SDK from the Microsoft Cognitive Services API to convert user's speech to text. We also used the turn.js library to create our virtual books. We then built our own algorithm to assess where the user's speech matches the text in the books they may be reading.

OcuLearn

Best Use of Google Cloud Services & Most Interesting Use of Data at HackDuke, November 2019

OcuLearn is a platform built to detect student engagement in online classrooms using Computer Vision and Machine Learning. The application collects real-time webcam data to process through Google Cloud Vision API providing principle axes head rotation measurements. These are run against our trained ML model that uses K-nearest neighbors algorithm and cluster analysis to reasonably assess the student's attentiveness.
