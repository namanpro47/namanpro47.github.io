Raptionary

Finalist Project at VTHacks, February 2017

Raptionary is a mobile application that gets you the lyrics of any song and provides an Urban Dictionary lookup tool that can give users definitions to any slang words or phrases used in a song. The program uses the jsoup library and provides lyrics and definitions by web crawling azlyrics.com and urbandictionary.com. The app is built on the Android platform. Download Raptionary for free on Google Play!

ESnap

VTHacks III, February 2016

ESnap is a mobile application that can provide a summary of an article or book page just from a screenshot or picture of a page. The program converts the image into text format using an Optical Character Recognition (OCR) API, and summarizes the text using a Natural Language Processing (NLP) API. Download ESnap for free on Google Play!
